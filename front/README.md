
  # PingTrackPong Monitoring System

  This is the code for the frontend of for PingTrackPong Monitoring System.

  ## Running the code

  1. *Prerequisite:* install the package manager `pnpm` following the indications at https://pnpm.io/installation. If you already have it, skip this step.
  2. Run `pnpm install` to install the dependencies.
  3. Run `pnpm run dev` to start the development server.
